## 新增事件

为了支持新功能，添加了以下事件（通过 `DeviceEventEmitter` 发送）：

1. **TaskConfirmationRequired** - 需要用户确认敏感操作
   - `taskId`: 任务ID
   - `step`: 步骤号
   - `message`: 确认消息
   - `actionType`: 操作类型

2. **TaskConfirmationResult** - 用户确认结果
   - `taskId`: 任务ID
   - `step`: 步骤号
   - `confirmed`: 是否确认

3. **TaskInteractRequired** - 需要用户选择
   - `taskId`: 任务ID
   - `step`: 步骤号
   - `message`: 提示消息
   - `options`: 选项列表

4. **TaskInteractResult** - 用户选择结果
   - `taskId`: 任务ID
   - `step`: 步骤号
   - `selection`: 用户选择

5. **TaskTakeOverRequired** - 需要用户接管
   - `taskId`: 任务ID
   - `step`: 步骤号
   - `message`: 接管消息

6. **TaskTakeOverCompleted** - 用户接管完成
   - `taskId`: 任务ID
   - `step`: 步骤号
