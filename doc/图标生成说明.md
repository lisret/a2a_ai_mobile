# 🎨 应用图标生成说明

## 📱 图标设计

已为应用设计了一个新的图标（`app_icon.svg`），设计特点：

- **主题**：AI 自动化助手
- **元素**：AI 机器人 + 手机 + 自动化齿轮
- **颜色**：蓝色渐变（#165DFF），与应用主题一致
- **风格**：现代、简洁、扁平化

## 🚀 快速生成图标（推荐方法）

### 方法一：使用 Android Asset Studio（最简单，推荐）

1. **打开 SVG 文件**：
   - 在浏览器中打开 `app_icon.svg` 查看图标设计

2. **转换为 PNG**：
   - 访问：https://cloudconvert.com/svg-to-png
   - 上传 `app_icon.svg`
   - 设置输出尺寸：1024×1024
   - 点击"转换"，下载 PNG 文件

3. **生成 Android 图标**：
   - 访问：https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html
   - 点击 "Image" 标签
   - 上传刚才转换的 1024×1024 PNG 文件
   - 选择 "Legacy Icon"（传统图标）或 "Adaptive Icon"（适配层图标）
   - 点击 "Generate" 生成所有尺寸
   - 下载 ZIP 文件

4. **替换图标文件**：
   ```bash
   # 解压下载的 ZIP 文件
   # 将 res/mipmap-*/ic_launcher.png 复制到：
   # AwesomeProject/android/app/src/main/res/mipmap-*/
   # 同样复制 ic_launcher_round.png 到各目录
   ```

### 方法二：使用 Node.js 脚本（自动化）

1. **安装依赖**：
   ```bash
   cd AwesomeProject
   npm install --save-dev sharp
   ```

2. **运行生成脚本**：
   ```bash
   npm run generate-icons
   ```

   脚本会自动：
   - 读取 `app_icon.svg`
   - 生成所有尺寸的 PNG 图标
   - 保存到对应的 `mipmap-*` 目录

## 📂 图标文件位置

图标需要放在以下位置：

```
AwesomeProject/android/app/src/main/res/
├── mipmap-mdpi/
│   ├── ic_launcher.png (48×48)
│   └── ic_launcher_round.png (48×48)
├── mipmap-hdpi/
│   ├── ic_launcher.png (72×72)
│   └── ic_launcher_round.png (72×72)
├── mipmap-xhdpi/
│   ├── ic_launcher.png (96×96)
│   └── ic_launcher_round.png (96×96)
├── mipmap-xxhdpi/
│   ├── ic_launcher.png (144×144)
│   └── ic_launcher_round.png (144×144)
└── mipmap-xxxhdpi/
    ├── ic_launcher.png (192×192)
    └── ic_launcher_round.png (192×192)
```

## ✅ 应用图标

生成图标后，重新构建应用：

```bash
cd AwesomeProject/android
./gradlew clean
cd ..
npm run android
```

安装到设备后，检查应用图标是否正确显示。

## 🎨 自定义图标

如果想修改图标设计：

1. 编辑 `app_icon.svg` 文件
   - 使用在线编辑器：https://boxy-svg.com/
   - 或使用 Adobe Illustrator、Inkscape

2. 重新生成图标（使用上述任一方法）

## 📝 注意事项

- ✅ 图标必须是正方形
- ✅ 重要内容应放在中心区域（边缘留 10% 安全区域）
- ✅ 圆形图标与普通图标相同，系统会自动裁剪
- ✅ 确保图标在不同尺寸下都清晰可见
- ✅ 推荐使用 Android Asset Studio 生成适配层图标（Adaptive Icon）
