# 快速部署指南

## 🚀 5 分钟快速开始

### 前提条件检查

```bash
# 检查 Node.js
node --version  # 应该 >= 18.0.0

# 检查 Java
java -version  # 应该显示 Java 17

# 检查 Android SDK
echo $ANDROID_HOME  # 应该显示 SDK 路径 (macOS/Linux)
# 或
echo %ANDROID_HOME%  # Windows
```

### 一键部署脚本

#### Windows (PowerShell)

```powershell
# 1. 安装依赖
npm install

# 2. 配置 Android SDK 路径 (编辑 android/local.properties)
# sdk.dir=C:\\Users\\<你的用户名>\\AppData\\Local\\Android\\Sdk

# 3. 启动 Metro
npm start

# 4. 在新终端运行 Android
npm run android
```

#### macOS / Linux

```bash
# 1. 安装依赖
npm install

# 2. 配置 Android SDK 路径 (编辑 android/local.properties)
# sdk.dir=/Users/<你的用户名>/Library/Android/sdk

# 3. 启动 Metro
npm start

# 4. 在新终端运行 Android
npm run android
```

## 📝 详细步骤

### 步骤 1: 安装依赖

```bash
cd AwesomeProject
npm install
```

**预计时间**: 3-5 分钟

### 步骤 2: 配置 Android SDK

创建或编辑 `android/local.properties`:

**Windows**:
```properties
sdk.dir=C:\\Users\\你的用户名\\AppData\\Local\\Android\\Sdk
```

**macOS**:
```properties
sdk.dir=/Users/你的用户名/Library/Android/sdk
```

**Linux**:
```properties
sdk.dir=/home/你的用户名/Android/Sdk
```

### 步骤 3: 启动 Metro Bundler

```bash
npm start
```

保持此终端窗口打开。

### 步骤 4: 连接设备或启动模拟器

#### 使用物理设备
1. 启用 USB 调试
2. 连接设备到电脑
3. 运行 `adb devices` 确认设备已连接

#### 使用 Android 模拟器
1. 打开 Android Studio
2. 启动 AVD Manager
3. 创建或启动模拟器

### 步骤 5: 运行应用

在新终端窗口中：

```bash
npm run android
```

## 🔧 常见问题快速修复

### Metro 端口被占用

```bash
# Windows
npm run kill:metro

# macOS/Linux
lsof -ti:8081 | xargs kill -9
```

### 清理构建缓存

```bash
# 清理 Metro 缓存
npm run start:reset

# 清理 Android 构建
cd android
./gradlew clean
cd ..
```

### 重新安装依赖

```bash
# 删除 node_modules
rm -rf node_modules  # macOS/Linux
# 或
rmdir /s node_modules  # Windows

# 重新安装
npm install
```
