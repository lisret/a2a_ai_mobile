## 系统要求

### 操作系统
- **Windows 10/11** (推荐)
- **macOS 10.15+** (用于 iOS 开发)
- **Linux** (Ubuntu 20.04+)

### 硬件要求
- **内存**: 至少 8GB RAM (推荐 16GB)
- **磁盘空间**: 至少 10GB 可用空间
- **CPU**: 多核处理器

### 软件要求
- **Node.js**: >= 18.0.0 (推荐 18.x 或 20.x LTS)
- **npm**: >= 9.0.0 (随 Node.js 安装)
- **Java JDK**: 17 (Android 开发必需)
- **Android Studio**: 最新稳定版
- **Android SDK**: API Level 21-34
- **Git**: 最新版本

## 环境安装

### 1. 安装 Node.js

#### Windows
1. 访问 [Node.js 官网](https://nodejs.org/)
2. 下载 LTS 版本 (推荐 18.x 或 20.x)
3. 运行安装程序，按默认选项安装
4. 验证安装：
   ```bash
   node --version
   npm --version
   ```

#### macOS
```bash
# 使用 Homebrew
brew install node@18

# 或下载安装包
# 访问 https://nodejs.org/ 下载安装
```

#### Linux (Ubuntu)
```bash
# 使用 NodeSource 仓库
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs
```

### 2. 安装 Java JDK 17

#### Windows
1. 访问 [Oracle JDK](https://www.oracle.com/java/technologies/downloads/#java17) 或 [OpenJDK](https://adoptium.net/)
2. 下载 JDK 17 (Windows x64)
3. 运行安装程序
4. 配置环境变量：
   - 新建系统变量 `JAVA_HOME` = `C:\Program Files\Java\jdk-17`
   - 在 `Path` 中添加 `%JAVA_HOME%\bin`
5. 验证安装：
   ```bash
   java -version
   javac -version
   ```

#### macOS
```bash
# 使用 Homebrew
brew install openjdk@17

# 配置环境变量
echo 'export PATH="/opt/homebrew/opt/openjdk@17/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

#### Linux (Ubuntu)
```bash
sudo apt update
sudo apt install openjdk-17-jdk
```

### 3. 安装 Android Studio

#### 所有平台
1. 访问 [Android Studio 官网](https://developer.android.com/studio)
2. 下载并安装 Android Studio
3. 启动 Android Studio，完成初始设置向导
4. 安装 Android SDK：
   - 打开 **SDK Manager** (Tools → SDK Manager)
   - 安装以下组件：
     - **Android SDK Platform 34**
     - **Android SDK Build-Tools 34.0.0**
     - **Android SDK Platform-Tools**
     - **Android Emulator**
     - **Intel x86 Emulator Accelerator (HAXM)** (Windows/macOS)

### 4. 配置 Android 环境变量

#### Windows
在系统环境变量中添加：
- `ANDROID_HOME` = `C:\Users\<用户名>\AppData\Local\Android\Sdk`
- 在 `Path` 中添加：
  - `%ANDROID_HOME%\platform-tools`
  - `%ANDROID_HOME%\tools`
  - `%ANDROID_HOME%\tools\bin`

#### macOS / Linux
在 `~/.zshrc` 或 `~/.bashrc` 中添加：
```bash
export ANDROID_HOME=$HOME/Library/Android/sdk  # macOS
# 或
export ANDROID_HOME=$HOME/Android/Sdk  # Linux

export PATH=$PATH:$ANDROID_HOME/platform-tools
export PATH=$PATH:$ANDROID_HOME/tools
export PATH=$PATH:$ANDROID_HOME/tools/bin
```

然后执行：
```bash
source ~/.zshrc  # 或 source ~/.bashrc
```

### 5. 安装 React Native CLI (可选)

```bash
npm install -g react-native-cli
```

**注意**: React Native 0.60+ 推荐使用 `npx react-native` 而不是全局 CLI。

### 6. 安装 Git

#### Windows
1. 访问 [Git 官网](https://git-scm.com/download/win)
2. 下载并安装 Git for Windows
3. 验证安装：
   ```bash
   git --version
   ```

#### macOS
```bash
brew install git
```

#### Linux (Ubuntu)
```bash
sudo apt install git
```

## 项目配置

### 1. 克隆或复制项目

```bash
# 如果使用 Git
git clone <项目地址>
cd Open-AutoGLM-ReactNative/AwesomeProject

# 或直接复制项目文件夹到目标位置
```

### 2. 安装项目依赖

```bash
# 进入项目目录
cd AwesomeProject

# 安装 Node.js 依赖
npm install

# 或使用 yarn (如果已安装)
yarn install
```

### 3. 配置 Android 本地属性

#### Windows
在 `AwesomeProject/android/` 目录下创建或编辑 `local.properties`：

```properties
sdk.dir=C:\\Users\\<用户名>\\AppData\\Local\\Android\\Sdk
```

#### macOS
```properties
sdk.dir=/Users/<用户名>/Library/Android/sdk
```

#### Linux
```properties
sdk.dir=/home/<用户名>/Android/Sdk
```

**注意**: 将 `<用户名>` 替换为你的实际用户名。

### 4. 配置 Metro Bundler 缓存 (可选)

如果遇到 Metro 缓存问题，可以清理缓存：

```bash
npm run start:reset
```

### 5. 安装 iOS 依赖 (仅 macOS)

```bash
cd ios
pod install
cd ..
```

## 运行项目

### 1. 启动 Metro Bundler

在项目根目录 (`AwesomeProject/`) 下：

```bash
npm start
# 或
npm run start
```

**注意**: 保持此终端窗口打开，Metro Bundler 需要持续运行。

### 2. 运行 Android 应用

#### 方法 1: 使用命令行

```bash
# 在新终端窗口中
npm run android

# 或指定设备
npm run android:emulator

# 查看可用设备
npm run android:list
```

#### 方法 2: 使用 Android Studio

1. 打开 Android Studio
2. 选择 `File → Open`
3. 选择 `AwesomeProject/android` 目录
4. 等待 Gradle 同步完成
5. 点击 `Run` 按钮 (绿色播放图标)

### 3. 运行 iOS 应用 (仅 macOS)

```bash
npm run ios
```

### 4. 调试

#### 查看日志
```bash
# Android
adb logcat *:S ReactNative:V ReactNativeJS:V

# iOS (macOS)
# 在 Xcode 中查看控制台
```

#### 打开开发者菜单
- **Android**: 摇动设备或按 `Ctrl+M` (模拟器) / `Cmd+M` (macOS)
- **iOS**: 摇动设备或按 `Cmd+D` (模拟器)

## 常见问题

### 1. Metro Bundler 启动失败

**问题**: `Error: listen EADDRINUSE: address already in use`

**解决方案**:
```bash
# Windows
npm run kill:metro

# 或手动查找并终止进程
netstat -ano | findstr :8081
taskkill /PID <进程ID> /F

# macOS/Linux
lsof -ti:8081 | xargs kill -9
```

### 2. Android 构建失败

**问题**: `SDK location not found`

**解决方案**:
- 检查 `android/local.properties` 文件是否存在
- 确认 `sdk.dir` 路径正确
- 在 Android Studio 中重新同步项目

**问题**: `BuildConfig not found`

**解决方案**:
- 清理构建缓存：
  ```bash
  cd android
  ./gradlew clean
  cd ..
  ```
- 重新构建项目

### 3. 依赖安装失败

**问题**: `npm ERR! network timeout`

**解决方案**:
```bash
# 使用国内镜像
npm config set registry https://registry.npmmirror.com

# 或使用 cnpm
npm install -g cnpm --registry=https://registry.npmmirror.com
cnpm install
```

### 4. Android 设备未识别

**问题**: `adb devices` 显示设备为 `unauthorized`

**解决方案**:
1. 在设备上允许 USB 调试
2. 检查 USB 连接模式 (选择"文件传输"或"MTP")
3. 重新连接设备
4. 在设备上确认"始终允许此计算机进行调试"

### 5. Java 版本不匹配

**问题**: `Unsupported class file major version`

**解决方案**:
- 确保使用 JDK 17
- 检查 `JAVA_HOME` 环境变量
- 在 Android Studio 中设置项目 JDK 版本

### 6. 原生代码编译错误

**问题**: Kotlin 编译错误

**解决方案**:
1. 清理构建：
   ```bash
   cd android
   ./gradlew clean
   ```
2. 检查所有导入是否正确
3. 确保包名与目录结构匹配
4. 重新同步 Gradle

### 7. 权限问题

**问题**: 应用无法获取权限

**解决方案**:
- 检查 `AndroidManifest.xml` 中的权限声明
- 在设备设置中手动授予权限
- 使用 `react-native-permissions` 库处理运行时权限

## 项目结构说明

### 目录结构

```
AwesomeProject/
├── android/              # Android 原生代码
│   ├── app/
│   │   └── src/main/java/com/awesomeproject/
│   │       ├── bridge/          # React Native 桥接模块
│   │       ├── service/         # Android 服务
│   │       ├── ui/              # UI 管理模块
│   │       ├── accessibility/  # 无障碍功能
│   │       ├── utils/           # 工具类
│   │       └── core/            # 核心应用类
│   ├── build.gradle
│   └── settings.gradle
│
├── ios/                  # iOS 原生代码 (macOS)
│   └── AwesomeProject/
│
├── src/                  # React Native 源代码
│   ├── core/            # 核心功能
│   ├── features/        # 功能模块
│   ├── navigation/      # 导航配置
│   └── shared/          # 共享组件和工具
│
├── package.json         # 项目依赖配置
├── tsconfig.json        # TypeScript 配置
├── babel.config.js      # Babel 配置
└── metro.config.js      # Metro Bundler 配置
```

### 关键配置文件

1. **package.json**: Node.js 依赖和脚本
2. **tsconfig.json**: TypeScript 编译配置
3. **babel.config.js**: Babel 转译配置
4. **android/build.gradle**: Android 构建配置
5. **android/app/build.gradle**: Android 应用配置
6. **android/local.properties**: Android SDK 路径 (需手动创建)

如果遇到问题：
