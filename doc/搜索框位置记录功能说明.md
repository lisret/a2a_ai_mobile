# 搜索框位置记录功能说明

## 功能概述

搜索框位置记录功能用于记录手机系统搜索框的位置描述信息，帮助 AI 模型快速定位搜索框，从而快速打开应用。

## 服务位置

**文件**：`src/services/SearchBoxPositionService.ts`

## 使用方法

### 1. 保存搜索框位置

```typescript
import { searchBoxPositionService } from '../services/SearchBoxPositionService';

// 保存搜索框位置描述
await searchBoxPositionService.saveSearchBoxPosition('搜索框在首页左侧页面的顶部');
```

### 2. 读取搜索框位置

```typescript
// 获取搜索框位置描述
const position = await searchBoxPositionService.getSearchBoxPosition();
if (position) {
  console.log('搜索框位置:', position);
} else {
  console.log('未找到搜索框位置记录');
}
```

### 3. 检查是否存在记录

```typescript
// 检查是否存在搜索框位置记录
const hasPosition = await searchBoxPositionService.hasSearchBoxPosition();
if (hasPosition) {
  console.log('已记录搜索框位置');
}
```

### 4. 删除搜索框位置

```typescript
// 删除搜索框位置记录
await searchBoxPositionService.deleteSearchBoxPosition();
```

## 在提示词中的使用

搜索框位置信息会自动添加到系统提示词中。当存在搜索框位置记录时，提示词会包含以下信息：

```
**系统搜索框位置信息**：
- 搜索框在首页左侧页面的顶部
- 提示：可以直接使用 Tap 操作点击搜索框，然后使用 Type 操作输入app名称进行搜索，找到app后点击进入。这样可以快速打开应用，无需在主屏幕上查找。
```

## 数据存储

- **存储方式**：使用 AsyncStorage 持久化存储
- **存储键名**：`@autoglm:search_box_position`
- **数据类型**：字符串（文本描述）

## 使用场景

1. **快速打开应用**：当需要打开应用时，AI 模型可以直接使用搜索框位置信息，快速定位并点击搜索框，输入应用名称进行搜索。

2. **减少探索时间**：通过预先记录搜索框位置，AI 模型无需在屏幕上探索查找搜索框，可以直接执行操作。

3. **提高任务效率**：特别是在需要频繁打开应用的场景下，可以显著提高任务执行效率。

## 示例

### 示例1：记录搜索框位置

```typescript
import { searchBoxPositionService } from '../services/SearchBoxPositionService';

// 用户发现搜索框在首页左侧页面的顶部
await searchBoxPositionService.saveSearchBoxPosition('搜索框在首页左侧页面的顶部');
```

### 示例2：在设置页面中使用

可以在设置页面中添加一个输入框，让用户输入搜索框位置描述：

```typescript
import { searchBoxPositionService } from '../services/SearchBoxPositionService';

// 在设置页面中
const [searchBoxPosition, setSearchBoxPosition] = useState('');

// 保存
const handleSave = async () => {
  try {
    await searchBoxPositionService.saveSearchBoxPosition(searchBoxPosition);
    Alert.alert('成功', '搜索框位置已保存');
  } catch (error) {
    Alert.alert('错误', '保存失败');
  }
};

// 读取
useEffect(() => {
  const loadPosition = async () => {
    const position = await searchBoxPositionService.getSearchBoxPosition();
    if (position) {
      setSearchBoxPosition(position);
    }
  };
  loadPosition();
}, []);
```

## 注意事项

1. **位置描述要准确**：位置描述应该清晰明确，例如：
   - ✅ "搜索框在首页左侧页面的顶部"
   - ✅ "搜索框在桌面首页的顶部中央"
   - ❌ "搜索框在某个地方"（太模糊）

2. **坐标信息（可选）**：如果需要更精确的位置，可以在描述中包含坐标信息，例如：
   - "搜索框在首页左侧页面的顶部，坐标约为 [100, 50]"

3. **更新位置**：如果搜索框位置发生变化（例如系统更新后），需要重新记录位置。

4. **自动集成**：搜索框位置信息会自动集成到提示词中，无需手动处理。
