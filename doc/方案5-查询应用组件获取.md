# 方案5：通过查询所有应用组件获取（最全面方案）

## 方案概述
通过查询 PackageManager 的所有组件（Activity、Service、Receiver、Provider）来反推出所有应用包名，这种方法可以获取到最全面的应用列表，包括那些没有启动入口但提供了服务的应用。

## 优势
- ✅ 获取最全面，包括所有有组件的应用
- ✅ 可以获取到纯服务类应用（如输入法、壁纸、通知监听等）
- ✅ 可以获取应用的所有组件信息
- ✅ 不依赖应用是否有启动入口

## 劣势
- ❌ 性能相对较慢（需要查询所有组件）
- ❌ 可能包含一些不应该显示的系统组件应用
- ❌ 需要过滤和去重

## 实现代码

### Kotlin 原生模块实现

```kotlin
@ReactMethod
fun getAllAppsByComponents(promise: Promise) {
    try {
        Log.d("AccessibilityActionModule", "开始通过查询组件获取所有应用")
        
        val pm = reactApplicationContext.packageManager
        val appsMap = Arguments.createMap()
        val packageSet = mutableSetOf<String>()
        
        // 方法1: 通过查询所有Activity获取包名
        try {
            val installedPackages = pm.getInstalledPackages(PackageManager.GET_ACTIVITIES)
            
            for (packageInfo in installedPackages) {
                try {
                    val packageName = packageInfo.packageName
                    val activities = packageInfo.activities
                    
                    // 如果有Activity，说明这是一个应用
                    if (activities != null && activities.isNotEmpty()) {
                        if (!packageSet.contains(packageName)) {
                            val appName = try {
                                pm.getApplicationLabel(packageInfo.applicationInfo).toString()
                            } catch (e: Exception) {
                                packageName
                            }
                            
                            val launchIntent = pm.getLaunchIntentForPackage(packageName)
                            
                            if (launchIntent != null) {
                                appsMap.putString(appName, packageName)
                            } else {
                                // 即使没有启动Intent，也记录（使用包名作为key）
                                appsMap.putString(packageName, packageName)
                            }
                            packageSet.add(packageName)
                        }
                    }
                } catch (e: Exception) {
                    // 忽略单个包
                }
            }
        } catch (e: Exception) {
            Log.w("AccessibilityActionModule", "通过Activity获取失败", e)
        }
        
        // 方法2: 通过查询所有Service获取包名（补充方法1可能遗漏的）
        try {
            val installedPackages = pm.getInstalledPackages(PackageManager.GET_SERVICES)
            
            for (packageInfo in installedPackages) {
                try {
                    val packageName = packageInfo.packageName
                    val services = packageInfo.services
                    
                    // 如果有Service，也记录这个包
                    if (services != null && services.isNotEmpty()) {
                        if (!packageSet.contains(packageName)) {
                            val appName = try {
                                pm.getApplicationLabel(packageInfo.applicationInfo).toString()
                            } catch (e: Exception) {
                                packageName
                            }
                            
                            // Service类应用通常没有启动Intent，使用包名作为key
                            appsMap.putString(packageName, packageName)
                            packageSet.add(packageName)
                        }
                    }
                } catch (e: Exception) {
                    // 忽略单个包
                }
            }
        } catch (e: Exception) {
            Log.w("AccessibilityActionModule", "通过Service获取失败", e)
        }
        
        // 方法3: 通过查询所有Receiver获取包名
        try {
            val installedPackages = pm.getInstalledPackages(PackageManager.GET_RECEIVERS)
            
            for (packageInfo in installedPackages) {
                try {
                    val packageName = packageInfo.packageName
                    val receivers = packageInfo.receivers
                    
                    if (receivers != null && receivers.isNotEmpty()) {
                        if (!packageSet.contains(packageName)) {
                            val appName = try {
                                pm.getApplicationLabel(packageInfo.applicationInfo).toString()
                            } catch (e: Exception) {
                                packageName
                            }
                            
                            appsMap.putString(packageName, packageName)
                            packageSet.add(packageName)
                        }
                    }
                } catch (e: Exception) {
                    // 忽略单个包
                }
            }
        } catch (e: Exception) {
            Log.w("AccessibilityActionModule", "通过Receiver获取失败", e)
        }
        
        // 方法4: 通过查询所有Provider获取包名
        try {
            val installedPackages = pm.getInstalledPackages(PackageManager.GET_PROVIDERS)
            
            for (packageInfo in installedPackages) {
                try {
                    val packageName = packageInfo.packageName
                    val providers = packageInfo.providers
                    
                    if (providers != null && providers.isNotEmpty()) {
                        if (!packageSet.contains(packageName)) {
                            val appName = try {
                                pm.getApplicationLabel(packageInfo.applicationInfo).toString()
                            } catch (e: Exception) {
                                packageName
                            }
                            
                            appsMap.putString(packageName, packageName)
                            packageSet.add(packageName)
                        }
                    }
                } catch (e: Exception) {
                    // 忽略单个包
                }
            }
        } catch (e: Exception) {
            Log.w("AccessibilityActionModule", "通过Provider获取失败", e)
        }
        
        // 方法5: 过滤系统组件（可选）
        // 移除一些明显的系统组件包名
        val systemComponentPatterns = listOf(
            "com.android.",
            "android.",
            "com.google.android.",
            "com.qualcomm.",
            "com.mediatek."
        )
        
        val filteredMap = Arguments.createMap()
        var filteredCount = 0
        
        val iterator = appsMap.keySetIterator()
        while (iterator.hasNextKey()) {
            val key = iterator.nextKey()
            val packageName = appsMap.getString(key)
            
            // 检查是否是系统组件
            val isSystemComponent = systemComponentPatterns.any { pattern ->
                packageName?.startsWith(pattern) == true
            }
            
            // 如果不需要过滤系统组件，或者不是系统组件，则添加
            // 这里可以根据需求决定是否过滤
            if (!isSystemComponent || true) { // 设置为true表示不过滤
                filteredMap.putString(key, packageName)
                filteredCount++
            }
        }
        
        Log.d("AccessibilityActionModule", "通过组件查询获取到 ${packageSet.size} 个应用包名，过滤后 ${filteredCount} 个")
        promise.resolve(filteredMap)
        
    } catch (e: Exception) {
        Log.e("AccessibilityActionModule", "通过组件查询应用列表失败", e)
        promise.reject("ERROR", "获取应用列表失败: ${e.message}", e)
    }
}
```

### 优化版本：批量查询所有组件

```kotlin
@ReactMethod
fun getAllAppsByComponentsOptimized(promise: Promise) {
    try {
        Log.d("AccessibilityActionModule", "开始优化版组件查询")
        
        val pm = reactApplicationContext.packageManager
        val packageSet = mutableSetOf<String>()
        val appsMap = Arguments.createMap()
        
        // 一次性查询所有组件信息
        val installedPackages = pm.getInstalledPackages(
            PackageManager.GET_ACTIVITIES or
            PackageManager.GET_SERVICES or
            PackageManager.GET_RECEIVERS or
            PackageManager.GET_PROVIDERS or
            PackageManager.GET_META_DATA
        )
        
        Log.d("AccessibilityActionModule", "获取到 ${installedPackages.size} 个包")
        
        for (packageInfo in installedPackages) {
            try {
                val packageName = packageInfo.packageName
                
                // 检查是否有任何组件
                val hasActivities = packageInfo.activities != null && packageInfo.activities.isNotEmpty()
                val hasServices = packageInfo.services != null && packageInfo.services.isNotEmpty()
                val hasReceivers = packageInfo.receivers != null && packageInfo.receivers.isNotEmpty()
                val hasProviders = packageInfo.providers != null && packageInfo.providers.isNotEmpty()
                
                // 如果有任何组件，就认为这是一个应用
                if (hasActivities || hasServices || hasReceivers || hasProviders) {
                    if (!packageSet.contains(packageName)) {
                        val appInfo = packageInfo.applicationInfo
                        
                        // 获取应用名称
                        val appName = try {
                            pm.getApplicationLabel(appInfo).toString()
                        } catch (e: Exception) {
                            packageName
                        }
                        
                        // 检查是否有启动Intent（优先显示）
                        val launchIntent = pm.getLaunchIntentForPackage(packageName)
                        
                        if (launchIntent != null) {
                            appsMap.putString(appName, packageName)
                        } else {
                            // 没有启动Intent，使用包名作为key
                            appsMap.putString(packageName, packageName)
                        }
                        
                        packageSet.add(packageName)
                    }
                }
            } catch (e: Exception) {
                Log.w("AccessibilityActionModule", "处理包失败: ${packageInfo.packageName}", e)
            }
        }
        
        Log.d("AccessibilityActionModule", "优化版组件查询获取到 ${packageSet.size} 个应用")
        promise.resolve(appsMap)
        
    } catch (e: Exception) {
        Log.e("AccessibilityActionModule", "优化版组件查询失败", e)
        promise.reject("ERROR", "获取应用列表失败: ${e.message}", e)
    }
}
```

## 组件类型说明

### Activity（活动）
- 应用的主要界面组件
- 有Activity的应用通常有用户界面
- 可以启动和交互

### Service（服务）
- 后台服务组件
- 可能没有用户界面
- 如：输入法服务、通知监听服务等

### Receiver（广播接收器）
- 接收系统广播的组件
- 可能没有用户界面
- 如：开机自启动、网络变化监听等

### Provider（内容提供者）
- 提供数据共享的组件
- 可能没有用户界面
- 如：联系人提供者、文件提供者等

## 适用场景
- ✅ 需要获取最全面的应用列表（包括服务类应用）
- ✅ 需要获取应用的组件信息
- ✅ 需要检测某些特定类型的应用（如输入法、壁纸等）
- ✅ 应用分析或安全审计场景

## 注意事项
1. **性能考虑**: 查询所有组件会比较慢，建议使用优化版本
2. **系统组件**: 某些系统组件可能不应该显示，需要过滤
3. **重复处理**: 需要正确处理去重，避免重复应用
4. **内存占用**: 如果应用很多，内存占用可能较大

## 优化建议
1. **批量查询**: 使用一次 `getInstalledPackages` 查询所有组件信息
2. **条件过滤**: 根据需求决定是否包含系统组件
3. **异步处理**: 如果应用数量很多，可以考虑异步处理
4. **缓存结果**: 组件信息变化不频繁，可以缓存结果

## 过滤规则建议

```kotlin
// 系统组件包名前缀（常见）
val systemPrefixes = listOf(
    "com.android.",
    "android.",
    "com.google.android.gms",
    "com.google.android.gsf",
    "com.qualcomm.",
    "com.mediatek.",
    "com.nxp.",
    "com.broadcom."
)

// 判断是否是系统组件
fun isSystemComponent(packageName: String): Boolean {
    return systemPrefixes.any { packageName.startsWith(it) }
}

// 判断是否是用户应用
fun isUserApp(appInfo: ApplicationInfo): Boolean {
    return (appInfo.flags and ApplicationInfo.FLAG_SYSTEM) == 0 ||
           (appInfo.flags and ApplicationInfo.FLAG_UPDATED_SYSTEM_APP) != 0
}
```
