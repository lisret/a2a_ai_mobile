# 文本输入问题分析

## 当前实现分析

### 原生模块实现位置
- **AccessibilityActionModule.kt** (第108-121行)：React Native 桥接层
- **AutoGLMAccessibilityService.kt** (第287-330行)：实际实现

### 实现逻辑

```kotlin
fun performTextInput(text: String): Boolean {
    // 1. 将文本复制到剪贴板
    val clipboard = getSystemService(CLIPBOARD_SERVICE) as ClipboardManager
    val clip = ClipData.newPlainText("text", text)
    clipboard.setPrimaryClip(clip)
    
    // 2. 查找当前焦点的输入框节点
    val rootNode = rootInActiveWindow ?: return false
    val focusedNode = rootNode.findFocus(AccessibilityNodeInfo.FOCUS_INPUT)
    
    if (focusedNode != null) {
        // 方法1: 直接设置文本（Android 6.0+）
        if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.M) {
            val arguments = Bundle().apply {
                putCharSequence(AccessibilityNodeInfo.ACTION_ARGUMENT_SET_TEXT_CHARSEQUENCE, text)
            }
            val success = focusedNode.performAction(AccessibilityNodeInfo.ACTION_SET_TEXT, arguments)
            if (success) {
                return true
            }
        }
        
        // 方法2: 粘贴操作
        val pasteSuccess = focusedNode.performAction(AccessibilityNodeInfo.ACTION_PASTE)
        if (pasteSuccess) {
            return true
        }
    }
    
    // 如果找不到输入框，返回 false
    return false
}
```

## 问题分析

### 问题1：需要先点击输入框
- **原因**：代码使用 `findFocus(AccessibilityNodeInfo.FOCUS_INPUT)` 查找当前焦点的输入框
- **影响**：如果用户没有先点击输入框，`focusedNode` 为 null，直接返回 false
- **解决方案**：
  1. 在执行 Type 操作前，确保先执行 Tap 操作点击输入框
  2. 或者在代码中自动查找并点击输入框（需要更复杂的逻辑）

### 问题2：输入框可能不支持 ACTION_SET_TEXT
- **原因**：不是所有输入框都支持 `ACTION_SET_TEXT` 操作
- **影响**：即使找到输入框，也可能无法直接设置文本
- **解决方案**：回退到粘贴操作或使用其他方法

### 问题3：粘贴操作可能失败
- **原因**：某些输入框可能不支持粘贴操作，或者需要特殊权限
- **影响**：即使剪贴板有内容，粘贴也可能失败
- **解决方案**：使用 ADB 回退（已实现）

### 问题4：缺少详细日志
- **原因**：当前日志不够详细，难以诊断问题
- **影响**：无法确定失败的具体原因
- **解决方案**：添加更详细的日志记录

### 方案2：自动查找输入框（可选）
如果找不到焦点输入框，尝试查找页面上的输入框并自动点击。

### 方案3：改进错误处理

使用 Android 的 InputMethodManager 来模拟键盘输入，但这需要更复杂的实现。

根据代码注释和实现，当前的最佳实践是：
1. **在执行 Type 操作前，先执行 Tap 操作点击输入框**
2. **如果无障碍服务失败，自动回退到 ADB 输入**（已实现）
3. **使用 ADB Keyboard**（如果可用）
