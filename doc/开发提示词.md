# AI智能工具APP - 开发提示词

## 项目概述

这是一款基于 **Open-AutoGLM** 框架的 AI 智能工具应用，能够帮助用户通过自然语言指令自动操作手机完成任务。应用采用 React Native 开发，支持 Android 和 iOS 平台。

**核心功能**：
- 用户输入自然语言指令（如"打开微信发消息给张三"）
- 应用通过 Open-AutoGLM 框架调用视觉模型理解界面
- 使用无障碍服务（Accessibility Service）执行自动化操作
- 循环执行：截图 → 视觉理解 → 输出操作指令 → 无障碍执行 → 循环

**参考框架**：https://github.com/zai-org/Open-AutoGLM

### 4. 核心功能实现

#### 4.1 模型配置数据结构

```typescript
interface AIModel {
  id: string;                    // 唯一标识
  name: string;                  // 模型名称
  apiUrl: string;                // API 地址
  apiKey: string;                // API 密钥（加密存储）
  description?: string;           // 模型描述
  modelName?: string;             // 模型标识（如 "autoglm-phone-9b"）
  createdAt: number;             // 创建时间戳
  updatedAt: number;             // 更新时间戳
}
```

#### 4.2 任务数据结构

```typescript
interface TaskStep {
  step: number;                  // 步骤序号
  action: string;               // 操作描述
  timestamp: number;             // 时间戳
  screenshotUri?: string;       // 截图路径（可选）
  actionDetails?: TaskAction;    // 操作详情
}

interface TaskOutput {
  steps: TaskStep[];             // 执行步骤列表
  finalScreenshot?: string;      // 最终截图
  summary?: string;              // 任务摘要
}

interface Task {
  id: string;                    // 任务ID
  modelId: string;               // 关联的模型ID
  instruction: string;          // 任务指令
  status: TaskStatus;            // 任务状态
  currentAction?: string;        // 当前操作描述
  error?: string;                // 错误信息
  output?: TaskOutput;           // 完整输出（包含所有步骤）
  createdAt: number;            // 创建时间戳
  completedAt?: number;          // 完成时间戳
}
```

#### 4.3 任务历史服务（TaskHistoryService）

**功能**：
- 保存任务记录（包括执行步骤）
- 按模型ID获取任务列表
- 获取任务详情
- 获取模型的最新任务
- 删除任务记录

**存储结构**：
- 所有任务：`@autoglm:tasks`
- 按模型索引：`@autoglm:tasks:model:{modelId}`

**实现要点**：
- 任务执行过程中实时保存步骤
- 任务完成/失败时保存最终状态
- 按创建时间倒序排列（最新的在前）

#### 4.4 Open-AutoGLM 集成

**API 调用流程**：
1. 用户输入任务指令
2. 应用调用 Open-AutoGLM API（使用配置的模型服务）
3. API 返回操作指令（点击坐标、输入文本等）
4. 通过无障碍服务执行操作
5. 循环执行直到任务完成

**API 请求格式**（参考 Open-AutoGLM）：
```typescript
interface AutoGLMRequest {
  model: string;                 // 模型名称
  messages: Array<{
    role: 'user' | 'assistant' | 'system';
    content: string | Array<{
      type: 'text' | 'image_url';
      text?: string;
      image_url?: { url: string };
    }>;
  }>;
  // 其他参数...
}
```

**实现要点**：
- 使用 `fetch` 或 `axios` 调用模型服务 API
- 处理 API 响应，解析操作指令
- 错误处理和重试机制
- 支持流式响应（如需要）

#### 4.5 无障碍服务实现（Android）

**核心要求**：
- 使用 Android Accessibility Service 执行自动化操作
- 支持点击、滑动、输入文本等操作
- 获取屏幕截图和界面元素信息

**Android 实现步骤**：

1. **创建无障碍服务类**（`AccessibilityService.kt`）：
```kotlin
class AutoGLMAccessibilityService : AccessibilityService() {
    // 实现点击操作
    fun performClick(x: Int, y: Int)
    
    // 实现滑动操作
    fun performSwipe(startX: Int, startY: Int, endX: Int, endY: Int)
    
    // 实现文本输入
    fun performTextInput(text: String)
    
    // 获取屏幕截图
    fun captureScreen(): Bitmap?
    
    // 获取界面元素信息
    fun getAccessibilityNodeInfo(): AccessibilityNodeInfo?
}
```

2. **配置 AndroidManifest.xml**：
```xml
<service
    android:name=".AccessibilityService"
    android:permission="android.permission.BIND_ACCESSIBILITY_SERVICE"
    android:exported="true">
    <intent-filter>
        <action android:name="android.accessibilityservice.AccessibilityService" />
    </intent-filter>
    <meta-data
        android:name="android.accessibilityservice"
        android:resource="@xml/accessibility_service_config" />
</service>
```

3. **创建服务配置文件**（`res/xml/accessibility_service_config.xml`）：
```xml
<accessibility-service
    xmlns:android="http://schemas.android.com/apk/res/android"
    android:description="@string/accessibility_service_description"
    android:packageNames="com.awesomeproject"
    android:accessibilityEventTypes="typeAllMask"
    android:accessibilityFlags="flagDefault|flagRetrieveInteractiveWindows"
    android:accessibilityFeedbackType="feedbackGeneric"
    android:notificationTimeout="100"
    android:canRetrieveWindowContent="true"
    android:canRequestFilterKeyEvents="true"
    android:canPerformGestures="true" />
```

4. **React Native 桥接**：
   - 使用 `react-native` 的 Native Modules 或 `@react-native-community` 相关包
   - 创建桥接模块连接 JavaScript 和 Kotlin 代码

**推荐库**：
- `react-native-accessibility-service`（如存在）
- 或自定义 Native Module

#### 4.6 任务执行流程

```typescript
async function executeTask(task: string, model: AIModel) {
  // 1. 初始化无障碍服务
  await AccessibilityService.initialize();
  
  // 2. 循环执行直到任务完成
  while (!isTaskComplete) {
    // 2.1 截图
    const screenshot = await AccessibilityService.captureScreen();
    
    // 2.2 调用 Open-AutoGLM API
    const action = await AutoGLMService.getAction(
      screenshot,
      task,
      model
    );
    
    // 2.3 执行操作
    switch (action.type) {
      case 'click':
        await AccessibilityService.performClick(action.x, action.y);
        break;
      case 'swipe':
        await AccessibilityService.performSwipe(
          action.startX, action.startY,
          action.endX, action.endY
        );
        break;
      case 'input':
        await AccessibilityService.performTextInput(action.text);
        break;
      case 'back':
        await AccessibilityService.performBack();
        break;
      // ... 其他操作
    }
    
    // 2.4 等待界面更新
    await sleep(500);
  }
}
```
