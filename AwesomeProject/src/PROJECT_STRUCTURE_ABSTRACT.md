## 📦 模块组织抽象图

## 🔌 接口抽象图

```
                    ┌─────────────────┐
                    │   Interface     │
                    │   (接口定义)    │
                    └────────┬────────┘
                             │
                ┌────────────┴────────────┐
                │                         │
                ▼                         ▼
    ┌──────────────────┐    ┌──────────────────┐

示例:
    ICapability → AccessibilityCapability
                → ADBCapability
    
    PromptProvider → OpenAutoGLMPromptProvider
                   → (未来: ClaudePromptProvider)
    
    DialogueParser → OpenAutoGLMParser
                   → (未来: ClaudeParser)
```

## 📐 设计模式应用

```
┌─────────────────────────────────────────────────────────┐
│                   设计模式应用                            │
│                                                           │
│  1. 工厂模式 (Factory Pattern)                            │
│     DialogueFactory                                      │
│     - getPromptProvider()                                │
│     - getParser()                                        │
│                                                           │
│  2. 策略模式 (Strategy Pattern)                          │
│     ICapability                                          │
│     - AccessibilityCapability                             │
│     - ADBCapability                                      │
│                                                           │
│  3. 模块模式 (Module Pattern)                            │
│     TaskExecutionEngine                                  │
│     - ScreenshotModule                                   │
│     - ModelInferenceModule                               │
│     - ActionExecutionModule                              │
│                                                           │
│  4. 观察者模式 (Observer Pattern)                         │
│     TaskStateModule                                      │
│     - emitTaskStarted()                                  │
│     - emitStepUpdate()                                   │
│     - emitTaskComplete()                                 │
│                                                           │
│  5. 适配器模式 (Adapter Pattern)                         │
│     CapabilityService                                    │
│     - 统一不同能力的接口                                  │
│                                                           │
│  6. 配置集中管理 (Configuration Management)            │
│     Constants Module                                     │
│     - 所有硬编码配置统一管理                              │
│     - 分类配置文件便于维护                                │
└─────────────────────────────────────────────────────────┘
```
